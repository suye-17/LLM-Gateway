## 🔄 6A工作流程

本项目采用6A工作流进行开发：
1. **Ask (询问)** - 深入理解需求和目标
2. **Analyze (分析)** - 分析技术方案和挑战
3. **Architect (架构)** - 设计系统架构和组件
4. **Arrange (安排)** - 制定详细开发计划
5. **Act (执行)** - 实施开发和测试
6. **Audit (审核)** - 代码审查和优化
